# Frontend — RAG Personal (Fase E)

Interfaz React + Vite. Consume únicamente la API FastAPI existente.

## Requisitos

- Backend en marcha: `uvicorn api.main:app --reload --host 127.0.0.1 --port 8000`
- Node.js 20+

## Desarrollo

```powershell
cd frontend
npm install
npm run dev
```

Abrir http://127.0.0.1:5173
